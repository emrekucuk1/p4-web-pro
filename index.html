YouTube API Bağlantısı

Gerçek P4 Otopark videolarını çekmek için ücretsiz bir YouTube Data API v3 key gerekiyor.

  • console.cloud.google.com adresine git
  • Yeni proje oluştur → "API ve Hizmetler" → "Kitaplık"
  • "YouTube Data API v3" ara → Etkinleştir
  • "Kimlik Bilgileri" → "API Anahtarı Oluştur" → Kopyala ve yapıştır
⚠ API key geçersiz veya bağlantı hatası. Tekrar dene.
